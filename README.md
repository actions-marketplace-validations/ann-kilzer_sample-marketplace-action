# sample-marketplace-action
sample marketplace action


This action prints an emoji based on the input. V1 only supports 😃 and 🐈️


## Inputs

## `keyword`

A string of text. Default `""`.

## Outputs

## `emoji`

The emoji output

## Example usage

uses: ann-kilzer/sample-marketplace-action@v1
with:
  keyword: 'cat'
